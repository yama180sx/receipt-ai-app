Issue1
Prisma直接参照をRepository化

現状

prisma.receipt.create()
prisma.receipt.findMany()

が各Serviceに散らばっています。

追加

repositories/
  receiptRepository.ts
  categoryRepository.ts
  productMasterRepository.ts
  memberRepository.ts

例

class ReceiptRepository {
  create()
  findById()
  findByImagePath()
}

効果

Prisma変更時

Prisma
↓
Drizzle

への移行コスト激減

優先度

★★★★★
