# Receipt AI Classifier (Tentative)

レシート画像から購入データを抽出し、家庭間での経費分類を行うアプリ。

## 技術スタック
- **Frontend**: React Native (Expo)
- **Language**: TypeScript
- **Backend**: Supabase (PostgreSQL, Auth, Storage, Edge Functions)
- **Infrastructure**: Docker
- **AI**: OpenAI API (GPT-4o / GPT-4o-mini)
